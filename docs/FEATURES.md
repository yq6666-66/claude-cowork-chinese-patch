本项目的目标是改善 Claude Desktop / Cowork 在中文环境下的可用性，重点覆盖用户高频接触的界面。

  - 项目入口
  - 推荐任务
- 左侧导航
  - 协作 / 代码
  - 新建任务
  - 项目
  - 计划任务
  - 自定义
  - 固定 / 最近
- 自定义页
  - Skills
  - connectors
  - plugins
  - 个人插件
  - 创建技能
  - 连接应用
  - 从 GitHub 安装技能
- 计划任务页
  - 页面说明
  - 空状态
  - 保持唤醒
  - 电脑唤醒时运行提示
- 设置页
  - 通用
  - 隐私
  - 功能
  - Claude 代码
  - 协作
  - 桌面应用通用设置
- 第三方推理配置
  - 连接、沙盒和工作区、遥测和更新
  - Gateway 凭据
  - API key、基础 URL、认证方式、额外请求头
  - 查看 JSON、本地应用、导出

- 个人资料下拉选项

  - 本地会话

  - 拉取请求

## 处理混合中英文

Cowork 有些界面会把一句话拆成多个 DOM 文本节点，导致只翻译一部分后出现类似：

```text
连接 your apps
创建 new skills
通用 desktop settings
计划任务 only run while your computer is awake.
连接ion
使用量 limits
```

因此翻译表同时包含完整句子和短片段规则，运行时翻译层会先匹配完整句子，再尝试替换片段。

## 未承诺覆盖

- Claude 远端新下发且翻译表中不存在的文案。
- 系统原生菜单和 Electron/Chromium 内置错误页面。
- 第三方插件自己的 UI 文案。
- 模型生成的对话内容。

新增遗漏文案时，请优先补充 `translations/zh-CN.json`。
